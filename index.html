JavaScript Платформер

Уровень: 1
Очки: 0
Жизни: 3
Монеты: 0

Управление:

A ← Влево
D → Вправо
W SPACE Прыжок
P Пауза
R Рестарт

Как играть:

  • Цель: Достигните зеленого флага, чтобы пройти уровень
  • Собирайте монеты (жёлтые круги) для получения очков
  • Избегайте красных врагов - они отнимают жизнь
  • Синие платформы - обычные, фиолетовые - отталкивают вверх высоко, а красные чуть-чуть
  • Соберите все монеты для бонусных очков

Советы:

  • Нажимайте R для быстрого рестарта
  • P ставит игру на паузу
  • Попробуйте собрать все монеты на уровне!
