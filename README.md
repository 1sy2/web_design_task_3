# JavaScript Платформер

![Game Screenshot](images/screenshot.png)

Полнофункциональная игра в жанре платформер, созданная на чистом JavaScript с использованием Canvas API.

## Демо

Игра доступна онлайн: [GitHub Pages](https://1sy2.github.io/web_design_task_3/)

## Использование нейросетей

- **Нейросеть**: DeepSeek
- **Время выполнения**: ~4 часа

## Функционал

### Основные возможности:
- Управляемый персонаж с плавной анимацией
- Движение влево/вправо ("A"/"D"/"Ф"/"В", стрелки)
- Прыжки ("W"/"Ц"/"Space") с поддержкой двойного прыжка
- Система гравитации и физики движения
- Коллизии с платформами и объектами
- Сбор монет для набора очков
- Враги с патрулированием территории
- Несколько уровней с возрастающей сложностью
- Условия победы/поражения
- Система жизней и подсчета очков

### Игровые элементы:
- **Платформы разных типов**:
  - Обычные (синие)
  - Прыгучие (фиолетовые) - отталкивают игрока высоко вверх
  - Опасные (красные) - отталкивают игрока чуть-чуть вверх
- **Монеты** - собирайте для очков
- **Враги** - избегайте красных кубов
- **Флаги** - цель каждого уровня

## Управление

| Клавиша | Действие | Альтернатива (русская раскладка) |
|---------|----------|-----------------------------------|
| **A** / **←** | Движение влево | **Ф** |
| **D** / **→** | Движение вправо | **В** |
| **W** / **Space** | Прыжок | **Ц** |
| **P** | Пауза / Продолжить | **З** |
| **R** | Рестарт уровня | **К** |

**Особенность:** Управление работает в любой раскладке клавиатуры!

## Запуск игры

### Онлайн версия:
1. Перейдите по ссылке: [GitHub Pages](https://1sy2.github.io/web_design_task_3/)

### Локальный запуск:
1. Скачайте или клонируйте репозиторий:
```bash
git clone https://github.com/1sy2/web_design_task_3.git
```

2. Пропишите в терминале:
```bash
# Python 3
python -m http.server 8000
```

3. Перейдите по адресу http://localhost:8000 в браузере

### Локальный запуск (альтернавный способ):

Скачайте или клонируйте репозиторий (как показано выше) и воспользуйтесь расширением "Live Server" в VS Code.
